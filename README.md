# Adventure-Works-Quarterly-Regional-Sales-Report
The goal of this project was to improve the accessibility and usability of the report for all users, including visually impaired users.

Key accessibility improvements implemented:

• Added descriptive report and chart titles
• Included alt text for visuals to support screen readers
• Added markers in line charts for better data point visibility
• Used shaded/high-contrast colors for improved readability
• Optimized tab order for keyboard navigation
• Applied an accessible theme with clearer fonts and better contrast

These enhancements help create a more inclusive and user-friendly reporting experience.

Tools Used:
Power BI | Data Visualization | Accessibility Design
